# How to execute?

To execute the code, follow these steps:
1. Clone the repository to your local machine. (git clone <repository_url>)
2. Navigate to the project directory. (cd <project_directory>)
3. Build the project using gradle: ./gradlew shadowJar
4. Browse the generated JAR file in the build/libs directory.
5. Run the JAR file using the command: java -jar <jar_file_name>.
